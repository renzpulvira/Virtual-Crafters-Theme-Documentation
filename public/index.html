# Virtual Crafters Wordpress theme

A starter wordpress theme with strong gulp configurations. Following the semi 7-1 architecture by Hugo giraudel and implementing best practices code.

The theme is inspired by the theme twentytwenty and underscores theme.

# Quick Start

Clone the repo

> git clone https://gitlab.com/vtl_gitlab/virtual-crafters-wordpress-theme.git

Install npm dependencies

> npm install # install npm dependencies

Start gulp

> gulp deploy # upload our base files

> gulp # Tell gulp to watch our files
